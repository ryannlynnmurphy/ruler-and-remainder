# Dorothy 

My chatbot. It's a machine that reads machines. Honestly, I made it because it's hard to tell what's real and what's marketing these days.

three tiers, kept rigorously apart — see [`METHOD.md`](METHOD.md):

- **Established** — checkable now, against sources you can pull yourself.
- **Exploratory** — plausible, coherent, untested at scale. Offered as a bet, with its conditions worn on the outside.
- **Speculative** — offered as geometry: a shape that organizes thinking, never load-bearing for a real system.

A speculative claim cannot be upgraded by how good it sounds, and an established one cannot be stretched past what the research actually shows. It's built for reading arguments honestly.
